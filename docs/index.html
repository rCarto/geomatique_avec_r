Chapitre 1 Préambule

Ce manuel est destiné tant aux utilisateurs de R souhaitant mettre en place traitements de l’information géographique qu’aux utilisateurs souhaitant utiliser R pour réaliser les taches qu’ils réalisent habituellement avec un SIG.
Les principales étapes du traitement de l’information géographiques y sont abordées. L’accent est porté sur le traitement des données vectorielles mais une partie est tout de même dédiée aux données raster.

Comment utiliser le manuel
Le projet RStudio contenant l’ensemble des données utilisées dans le manuel est disponible ici. Une fois le dossier décompressé il est possible de tester l’ensemble des manipulations proposées dans le projet RStudio.
